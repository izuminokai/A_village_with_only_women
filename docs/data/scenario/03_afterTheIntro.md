title: 導入後の流れ
---

- 通常ルート
    1. [婦人]に話しかける
    2. [婦人]が[施設.村長の家]へと案内してくれる
    3. [村長]と会話
        - 村に来た理由を聞かれる
            - 「すぐに村を出る」と答える
                - [同行者]が反対する。これを説得できれば村を出られる（ノーマルエンド）
            - 「村へ定住したい」と言う
                - [シナリオ変数.定住希望] := True
                - PCが信用に足るかどうか、3日間かけて試すことになる
                - 「定住」という単語が入っていなくても、村に住むことが前提となるならそれは定住希望と見なされる。「ハーレムを作りたい」など
                    - [村長]「それはつまり、この村に住みたいということですか？」
        - 3日間だけ滞在する許可をくれる
    4. 夜、[施設.村長の家]の客間に泊めてもらう
        - [村長]が手料理を運んできてくれる
        - [同行者]と会話した場合
            - [同行者]「村長さんは何か隠している気がします」
    5. ->[村探索](04_villageExploration.html)
- [同行者]に村へ来た理由を尋ねた場合
    - -> [NPC設定.同行者.目的]
- [施設.村長の家]へ行くことを拒否した場合や、泊まることを拒否した場合
    - 夜は[村探索.野宿]となる
    - [施設.村長の家]以外に「泊めてほしい」と頼んだ場合
        - 断られる
            - 外部の人間は、[施設.村長の家]以外に泊めてはいけない決まり
- [村長]に「宿泊費を払う」と言った場合
    - 固辞される
    - [村長]の[好感度]+=4
