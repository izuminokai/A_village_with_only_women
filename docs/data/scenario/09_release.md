title: 解放
---

1. 神社に到着
    - [巫女]にお礼を言われる
2. ここから先は、追ってくる[副長]から逃げながらの進行となる
    - [巫女]「あまり呑気にしてはおれん。副長のやつに勘付かれればおしまいじゃ」
      [副長]が村の殺し屋のような役割も担っていることを、[巫女]から聞かされる
    - 仮に倒れたNPCが[副長]だったとしても、無理やり起き上がってPCを阻止しようとしてくる
3. 大岩の前に案内され、動かすように言われる
    - [能力値]は体力を使う
    - 大成功
        - すぐに岩をどかせる
            - 疲労（軽度）
        - 協力NPCがいる場合
            - 「私はここで見張っていますから、早く行ってください！」（NPCの口調に合わせて修正する）
            - [残り時間] := 40分（[副長]♥ありなら80分）
        - 協力NPCがいない場合
            - [残り時間] := 20分（[副長]♥ありなら40分）
    - 部分成功
        - 岩をどかすのに時間がかかる
            - 疲労（重度）
        - どかし終えた時、遠くから[副長]が走ってくるのが見える
        - 協力NPCがいる場合
            - 足止めしてくれる
            - [残り時間] := 20分（[副長]♥ありなら40分）
        - 協力NPCがいない場合
            - [残り時間] := 10分（[副長]♥ありなら20分）
    - 失敗
        - 岩をどかし終える前に、[副長]が襲いかかってくる
        - 協力NPCがいる場合
            - 協力NPCが足止めしている間になんとか岩をどかす
            - 疲労（重度）
            - [残り時間] := 10分（[副長]♥ありなら20分）
        - 協力NPCがいない場合
            - 取り押さえられる
            - ->[「協力要請」や「解放」のフェーズで捕まった場合](ifCaught.html)
4. 岩をどけると、真っ逆さまに落ちる穴がある。これを[巫女]と一緒に下っていく
    - [能力値]は体力を使う
    - 急いで降りるか、慎重に降りるかで結果が変わる
    - 大成功
        - 急いで降りた場合
            - 負傷（軽度）の[状態異常]になる
        - 慎重に降りた場合
            - [残り時間] -=10分
    - 部分成功
        - 急いで降りた場合
            - 負傷（軽度）の[状態異常]になる
            - [残り時間] -=10分
        - 慎重に降りた場合
            - [残り時間] -=20分
    - 失敗
        - 急いで降りた場合
            - 負傷（重度）の[状態異常]になる
            - [残り時間] -=10分
        - 慎重に降りた場合
            - [残り時間] -=30分
5. 降りた先には[村長]がいる
    - [村長]「驚きましたか？　実はここは、私の部屋と地下で繋がっているのです」
      [村長]「あなたが通ってきた場所は、龍脈を通すための穴。本来は人が通る場所ではありません」
    - [状態異常]が負傷の場合、ここで村長に心配される
        - [村長]「その怪我、大丈夫ですか？　今すぐに治療したほうが良いのでは……」
6. 立ちはだかる村長を説得するか、力ずくでどける
    - [村長]♥ありの場合、邪魔をしてこないので説得は不要
    - 説得
        - [能力値]は交渉を使用する
        - [状態異常]が少し影響する
        - 説得1回あたり : [残り時間]-=5分
        - 大成功
            - 渋々だが納得して、引き下がってくれる
            - この時点で[残り時間]が0分になったとしても、[村長]が[副長]のことを押し留めてくれる
        - 部分成功
            - 納得しない。説得は継続する
        - 失敗
            - 交渉決裂。力ずくしかない
    - 力ずく
        - [能力値]は体力を使用する
        - [状態異常]が大きく影響する
        - 大成功
            - 残り時間 -=10分
            - この時点で[残り時間]が0分以下の場合
                - [副長]が追いついて、取り押さえられる
                - ->[「協力要請」や「解放」のフェーズで捕まった場合](ifCaught.html)
        - 部分成功
            - 残り時間 -=20分
            - この時点で[残り時間]が0分以下の場合
                - [副長]が追いついて、取り押さえられる
                - ->[「協力要請」や「解放」のフェーズで捕まった場合](ifCaught.html)
        - 失敗
            - 逆に取り押さえられる
            - ->[「協力要請」や「解放」のフェーズで捕まった場合](ifCaught.html)
7. 神鏡を破壊する
    - まばゆい光が辺りを包む
      [村長]「これで終わり、ですね……」
      [巫女]「いいや、まだ終わっとらん。おい、お主」
      [巫女]がPLに言う
      [巫女]「その鏡の破片で、わしを刺し殺せ」
8. [巫女]が自分を殺すよう懇願してくる
    - [巫女]「神鏡はあくまでも、わしを縛る鎖でしかない。不老不死の力の根源は、わしそのものじゃ。わしが存在する限り、また同じことが起こる」
      [巫女]「本来、人の子にわしは殺せんが……その鏡は神器じゃ。欠片を胸に突き立てれば、神殺しとて可能じゃろうて」
      [巫女]「時が経てば、わしも本来の力を取り戻してしまう。今が最後のチャンスなのじゃ」
      [巫女]「じゃから頼む。わしを、殺してはくれんか？」
9. [巫女]を殺しても、殺さなくても、どちらでも可
    - 殺害しても[シナリオ変数.殺害]は**付与されない**
