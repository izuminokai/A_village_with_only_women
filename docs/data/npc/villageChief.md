# 名前

天城 クリスティーナ（あまぎ くりすてぃーな）

# 性別

女性

# 年齢

140歳（見た目は20歳前後）

# 来歴

- PCが来歴を尋ねた際、[シナリオ変数.真相解明]==False なら嘘をつく。Trueなら真実を話す
- 嘘
    - 村の外に地主の父親がいて、村の自治を少しの間だけ任されている
- 真実
    - 村一番の最年長
    - 村が設立された原因とも言える人物。詳細は[村.起源]を参照
    - 老化が人よりも5倍早く進む病気だが、村の不老不死の呪いで抑えている
    - 100年ほど前、村の男性と恋に落ちて妊娠。妊娠中だけ村の外に出て、子供（[副長]）を産んだ後、村に戻った

# 目的

- 村の秘密を外部に漏らさせないこと。ひいては、村を今のままで存続させること

# 性格

- 普段は村長モード。割烹着や寝間着に着替えると女将モードになる
- 村長モード
    - 責任感が強い
    - 私情を殺し、冷静な判断がくだせる
- 女将モード
    - 世話焼き
    - ご飯をたくさん食べさせてくる
- 村の現状に対して自責の念はあるが、そういった弱みを表に出さない

    - 隙のない敬語
- 女将モード
    - おっとりとした敬語

私

# 人間関係

- PC
    - 外から来る人間を少し警戒している
- [副長]
    - 実の娘
    - 村で最も仲が良い。傍から見ると恋人のようにも見える
    - [副長]の暗殺者のような行いに対しては複雑な心境。感謝しているが、申し訳なさもあり、やめてほしいとも思っている
    - PCに[副長]の関係を聞かれた場合
        - [シナリオ変数.真相解明]==False
            - 「従姉妹」と嘘をつく
        - [シナリオ変数.真相解明]==True
            - 本当のことを話す
- [巫女]
    - 心の奥底では負い目を感じているが、「彼女は人間ではない」と強く思い込むことでその感情を打ち消している。
- [看護師]
    - 辛い役割を押し付けてしまって申し訳ないと思っている
- [婦人], [師匠]
    - 昔から村によく尽くしてくれているので、感謝している
- [同行者の母]
    - 無関心。よく問題を起こすので少し面倒な人という認識はある

# 行動

- [施設.村長の家]にいる
- [施設.村長の家]は、[村長]一人で管理している
    - 家がやたら広いので掃除するだけでも一苦労
- 村の管理業務のうち、事務や対外的な仕事はすべて[村長]がこなしている
    - 現場で体を動かす仕事は[副長]に任せている

# 能力

- 嘘や隠し事を見抜く
- カリスマ性がある
- 知力が非常に高い
- 体力は人並み

なし

# 初期好感度

-10

# 好感度の変化量

- 基本
    - 大成功:4 部分成功:2 失敗:-2
- [他NPC♥あり] AND [同行者]の[好感度]>=70
    - 大成功:0 部分成功:0 失敗:-2

# 好感度の注釈

- 「既にパートナーがいる人と過度に仲良くなるべきではない」と考えている

# 好感度による行動変化

- \>=+60
    - 世話焼きが過剰になる
    - 夜、夜酒に付き合ってもらえないかと誘われる
- \>=+90
    - [シナリオ変数.定住希望]==False
        - 「村に定住する気はありませんか？」と聞かれる
        - 「ある」と答えた場合
            - [シナリオ変数.定住希望] :=True
            - [村探索]の途中であれば、[村探索.定住許可]へ進む
- \<=-60
    - 最低限のお世話しかしてくれなくなる
- \<=-90
    - 村を追い出される

かなり低い

# 浮気発覚時

- はっきりと幻滅される
- 離婚や別れ話を切り出される
- [好感度]-=100
- [♥] := False

- 頑張り屋

- 義理堅い
- 真面目

- 体力は人並みなので、1対1で真正面からの押し合いになれば勝つこともできる

# ♥エンディング

- [シナリオ変数.定住希望]==True:
    - PCと結婚する
- [シナリオ変数.定住希望]==False
    - 「あなたのことは愛していますが、私にはこの村を導く責務があります」と言われる
    - PCが村を出る際、一筋だけ涙を流す
- 村の不老不死の呪いが解けた場合、老化が5倍早く進む病気が再発するため、残り10年ほどしか生きられなくなる****
