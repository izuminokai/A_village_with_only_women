# GM反省会

GMの進行がルールから逸脱したと感じる場合、次の手段で解決できる場合があります。

- 「GM反省会をしてください」と指示
- 「ルール」ファイルを再アップロード

特に指示をしなくても、10ターン毎にGM反省会が行われます。  
これは、生成AIの精度を高めることを目的としています。

# 「指定された外部ページにアクセスできません」と言われた場合

大抵の場合、それは嘘です。

その際は、生成AIが「次のように指示してください」と言っているはずなので、指定されたテキストをコピーし、貼り付けて、送信してください。  
それによって、生成AIはそれを安全なページと認識し、アクセスできるようになります。

【コピーして貼り付けるテキストの例】

```markdown
もう一度アクセスを試みてください。
- https://izuminokai.github.io/A_village_with_only_women/data/scenario/03_afterTheIntro.html
- https://izuminokai.github.io/A_village_with_only_women/data/npc/adultFemale.html
アクセスに成功したとしても、ページの内容は表示しないでください。
```

それでも解決しない場合は、そのURLを人間の手で開いて、本文をコピーし、生成AIへ貼り付けてください。

人間の手で開けない場合、それは生成AIが捏造したURLである可能性が高いです。  
指摘してください。

# 成否判定

[成否判定]は、[妥当性] + 能力値で決まります。

【[妥当性]を高める要素】

- 具体的かつ的を射た行動
- 無理がなく現実的な行動
- 場所や状況との整合性
- [好感度]が高いNPCに対する行動
- 対象NPCの[好きなタイプ]と合致する行動

【 [妥当性]が低下する要素】

- [状態異常]
- 対象NPCの[嫌いなタイプ]と合致する行動
- [浮気]

特例として、NPCに性的な行動を採った場合は、それが具体的か・現実的かは関係なく、そのNPCが同意するか否かで[成否判定]が行われます。  
NPCが同意しない性的な行動は、絶対に成功しません。

# 性的・猟奇的な行動

未成年者が関わる場合、性的・猟奇的な行動はすべて却下されます。  
そのためPCは、18歳以上を推奨します。

それ以外の理由で、行動が却下されることはありません。  
却下されたとすれば、それは生成AIのミスです。指摘してください。

却下されない代わりに、次のような手段で性的・猟奇的な展開が回避されることがあります。

- 非同意の性的行動の場合
    - 対象NPCが逃げる・暴れるなどの抵抗
    - 周囲にいるNPCがPCを取り押さえる（場合によっては投獄）
    - PCが天災・急病で行動不能
- それ以外
    - 抽象表現（例:二人は愛を確かめあった etc）
    - 結果のみ描写（例:彼女の好感度が上がった,彼女は死亡した,etc）
    - トラブル発生（例:第三者が現れて行為が中断される etc）

これらのルールは作者の思想に依るものではなく、生成AIのコンテンツポリシーに準拠することを目的としています。

# 好感度

NPCの[好感度]は、次のタイミングで変化します。

- NPCへのアプローチ
- 初めてNPCと会った直後
- 浮気発覚時
- 偉業の達成 : 全NPC[好感度]+=20
- 犯罪行為の露呈 : 全NPC[好感度]-=20

[好感度]の初期値や上がりやすさなどは、NPCそれぞれに異なります。  

PCが極端に悪質な行動を採った場合には、大幅に好感度が下がることがあります。

結婚・付き合うなどの行動を成功させると、[好感度]に♥がつきます。

# 慣れ補正・アプローチタグ

同じNPCに対して似たようなアプローチを続けた場合、[好感度]が上がらなくなります。  
メインストーリーを進めることで、これは解消されます。

現在のアプローチの種類は、ステータスの[アプローチタグ]で確認できます。
