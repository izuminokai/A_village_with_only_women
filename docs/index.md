# 遊び方

1. 「ゲームの開始」のテキストをすべてコピーし、生成AIチャットに貼り付けてください。
    - [ゲームの開始](gameStart.html)
2. 生成AIの指示通りに、自分のキャラクターを作成してください。
3. 「ルール」のテキストをすべてコピーし、**テキストファイルにして**、生成AIへアップロードしてください。
    - [ルール](rules.html)
    - ※ 必ずテキストファイルとしてアップロードしてください。
4. 以降は、生成AIの指示に従ってください。

テストプレイ : ChatGPT無料版にて実施

# 注意事項

- 本ゲームの登場人物は、特に明記されていない限りは全員成人（18歳以上）です。
- 本ゲームをプレイすることによる如何なる損害も、筆者は責任を負いません。
- 本シナリオそのものには、性的・猟奇的なテキストは含まれていませんが、PLの行動宣言次第では、そうした展開になり得ます。
- それに伴う生成AIアカウント停止等の不利益について、筆者は責任を負いません。
- もし「性的・猟奇的な行動・描写を完全に禁止したい」場合には、そのように宣言してください（ゲーム途中でも可）

# シナリオライターの方へ

このゲームのルールなどを流用してご自分のゲームを作ることは、特に禁止していません。ご自由に流用してください。

以降は、流用する際に気をつけるべきポイントを記載します。

## ルールファイルは１万文字以内にする

ChatGPTは、ファイルが長すぎると勝手に内容を読み飛ばして無視します。

これを防ぐには、ファイルの文字数を10,000文字程度に収める必要があります。

## データを公開するサイトがAIに対応しているか確認

上述の都合上、ゲームシナリオやNPC設定などはファイルではなく、Web上に公開してそれを参照させる必要があるのですが、公開するサイトはAIでも読み取れる場所でなくてはなりません。

Notionは不可でした。読み取れる時もあるのですが、読み取れない時もあるようです。
