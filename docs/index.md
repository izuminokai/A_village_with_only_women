# 遊び方

1. [ゲームの開始](https://raw.githubusercontent.com/izuminokai/A_village_with_only_women/refs/heads/main/docs/gameStart.md) の**テキストをすべてコピー**し、生成AIチャットに貼り付けてください。
2. 生成AIの指示通りに、自分のキャラクターを作成してください。
3. [ルール](https://raw.githubusercontent.com/izuminokai/A_village_with_only_women/refs/heads/main/docs/rules.md) を**ファイルとして保存**し、生成AIへアップロードしてください。
4. 以降は、生成AIの指示に従ってください。

テストプレイ : ChatGPT無料版にて実施

# シナリオライターの方へ

このゲームのルールなどを流用してご自分のゲームを作ることは、特に禁止していません。ご自由に流用してください。

以降は、流用する際に気をつけるべきポイントを記載します。

## ルールファイルは１万文字以内にする

ChatGPTは、ファイルが長すぎると勝手に内容を読み飛ばして無視します。  
これを防ぐには、ファイルの文字数を10,000文字程度に収める必要があります。

## データを公開するサイトがAIに対応しているか確認

上述の都合上、ゲームシナリオやNPC設定などはファイルではなく、Web上に公開してそれを参照させる必要があるのですが、公開するサイトはAIでも読み取れる場所でなくてはなりません。

- Notion : 不安定。読み取れる時もあるが、読み取れない時もある
- FC2 : 不可。503エラー
- GitHub Page : 可

## ファイル形式はmdかjsonにする

通常のテキストは、生成AIには読みにくいようです。mdにしたほうが精度が上がります。

jsonにすると更に精度が上がる感覚があります。  
ただ、jsonはデータレイアウトがある程度固定されている場合に使うべきものですから、すべてjsonにすれば良いというわけではないと思われます。
