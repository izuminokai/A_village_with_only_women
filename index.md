# 生成AI TRPG「女性しかいない村」

## 推奨プレイ環境

- ChatGPTの無料版にてテスト実施
- **考える（思考）モードを強く推奨**

## 遊び方

1. [ゲームの開始](https://raw.githubusercontent.com/izuminokai/A_village_with_only_women/refs/heads/main/gameStart.md)
   のテキストをすべてコピーし、生成AIチャットに貼り付けてください。
2. 生成AIの指示通りに、自分のキャラクターを作成してください。
3. <a href="rules.txt" download="ルール.md">ルール</a> をダウンロードし、生成AIへアップロードしてください。
4. 以降は、生成AIの指示に従ってください。

## ストーリー

あなたはある日、「女性しかいない村」が実在するという噂を耳にする。  
その村は百年以上も前から在るにも関わらず、人口が減ることのない、不思議な村だと言う。  
興味をそそられ調べていると、不意に１本のメールが届く。

    私はその村の在り処を知っています
    良ければ一緒にその村へ行ってみませんか？

誘いに乗ったあなたは、そのメールの主と共に、  
山奥にあるというその村へと向かった――
