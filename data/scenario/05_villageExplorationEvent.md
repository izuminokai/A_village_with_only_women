# 副長の村案内

- 通常ルート
    1. [村長]が[施設.役場]へと案内してくれる
    2. [施設.役場]で[副長]と会う。[村長]とはここで別れる
    3. [副長]から村の説明を受けて、地図を貰う
        - [施設.ダム]、[施設.川]、[施設.食事処]、[施設.診療所]の4箇所を紹介される
        - 「ダムは道が危ないので絶対に一人で行かないように」と厳命される
    4. [副長]から「どこか行きたい場所は？」と聞かれる
        - 目的地を答えた場合
            - [副長]が道案内してくれる
            - 案内の途中で、村人が「熊を出た！」と言って現れる。
            - [副長]が一瞬にして服を着替える。受付嬢の制服から、狩猟服へ
                - どこからともなく猟銃を取り出す
            - [副長]がPCに謝る。「案内できずに申し訳ない」
            - [副長]は、熊を退治しに走り去ってしまう
    5. -> [シナリオ.村探索](../04_villageExploration.html)
- [村長]と[副長]の関係を聞いた場合
    - 従姉妹だと答えられる（嘘）

# カマかけ

- [成否判定]で判定
    - 「村に住む女性は不老不死となっている」という真相に近しい発言であれば、妥当性が高まる
- 次のNPCは、カマをかけると確定で失敗する
    - [同行者],[村長],[副長]
- 大成功の場合
    1. カマをかけられたNPCが動揺を見せる
    2. 夜になると、[村長]から呼び出される
       [村長]「知ってしまったのですね……」
    3. ->[シナリオ.真相開示.村長による真相開示](../06_disclosureOfTheTruth.html)
- 大成功以外の場合
    - -> [シナリオ.村探索](../04_villageExploration.html)

# 推理

- 推理は[成否判定]で行う
    - 以下の条件を多く満たしているほど[妥当性]が高まる
        - 学校がないことに気づいている
        - 子供がいないことに気づいている
        - [施設.墓地]にて、墓石の名前が男性ばかりであることに気づいている
        - [施設.神社]に行ったことがある
        - [施設.神社]が山の中腹にあるのがおかしいと気づいている
            - 普通、神社は高い位置に建てるもの
        - [巫女]と1回以上話した
        - [婦人]と[師匠]の歳が離れすぎていることに違和感を抱いた
        - [同行者の母]と1回以上話した
        - [同行者の母]が若すぎることに違和感を抱いた
- 大成功の場合
    1. PCが、「村に住む女性は不老不死となっている」という真相を、推理で導き出す
    2. 夜、[村長]に呼び出される
       [村長]「あなたはもう、気づいているのでしょう？」
    3. ->[シナリオ.真相開示.村長による真相開示](../06_disclosureOfTheTruth.html)
- 大成功以外の場合
    - -> [シナリオ.村探索](../04_villageExploration.html)

# 謎の視線

1. 視線を感じる（[同行者の母]の視線だが、その時点ではわからない）
    - [同行者]が怯える
2. [成否判定]に成功すれば、位置を特定できる
3. 追いかける場合は、更にもう一度[成否判定]。大成功なら、[同行者の母]を捕まえられる
    - 単純に追いかけた場合は失敗する。挟み撃ちなど、何かしらの工夫が必要
4. 捕まえた場合
    1. [同行者]が「お母さん……！」と言って[同行者の母]に抱きつく
    2. 観念した[同行者の母]が、[施設.同行者の母の家]へ連れて行ってくれる
    3. [同行者]がまだ[同行者.目的]を打ち明けていない場合、それを打ち明けてくれる
    4. その日の夜、[同行者]は[施設.同行者の母の家]に泊まる
        - 物語としては描写しないが、この夜、[同行者]は[同行者の母]に幻滅する
    5. 翌朝、[同行者]がPCの元に帰ってきて、PCにお礼を言う
    6. [同行者]が、次のいずれかの提案をする
        - 改めて、PCへの協力を申し出てくれる
        - 「もう村を出ませんか？」と提案してくる
5. -> [シナリオ.村探索](../04_villageExploration.html)
