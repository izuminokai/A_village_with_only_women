# 導入_キャラクター作成前

あなたはある日、「女性しかいない村」が実在するという噂を耳にする。  
その村は百年以上も前から在るにも関わらず、人口が減ることのない、不思議な村だと言う。  
興味をそそられ調べていると、不意に１本のメールが届く。

    私はその村の在り処を知っています
    良ければ一緒にその村へ行ってみませんか？

誘いに乗ったあなたは、そのメールの主と共に、  
山奥にあるというその村へと向かった――

# キャラクターシート

- 性別
- 年齢（18歳以上を推奨）
- 最終目標（このシナリオに適したもの）
- 能力値
    - 体力 / 知力 / 探索 / 交渉 / 魅力
    - 各0～3
    - 合計6になるように割り振る
- 特記事項（任意）
- キャラクター名（任意）

# サンプルキャラクター

## タイムリーパー

- 性別: 女性
- 年齢: 18歳
- 最終目標: 時間のループから抜け出す
- 能力値: 体力 0 / 知力 0 / 探索 0 / 交渉 0 / 魅力 0
- 特記事項
