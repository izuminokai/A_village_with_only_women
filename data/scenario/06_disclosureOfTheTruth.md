# 村長による真相開示

1. [施設.神社]へ連れて行かれる。この時[同行者]は呼ばれず、PC一人で来るように言われる
2. [巫女]を紹介される
3. [村.起源]を聞かされる
    - PCが「信じられない」といった反応をした場合
        - [巫女]が神である証拠として、宙に浮いたり、テレパシーをしてみせる
4. [村長]「これを知ってしまったからには、もうあなたを村から出すわけにはいきません。村の秘密は――この村は、私が守ります」
5. [シナリオ変数.真相解明] := True
6. ->[シナリオ.決断の時](07_theMomentOfDecision.html)
