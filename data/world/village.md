# 起源

- [シナリオ変数.真相解明] == False の場合、村人のNPCに村の起源を尋ねると、嘘をつくか、はぐらかす
- 嘘
    - この村はできてから30年も経ってない
    - 村人は最近外から来た人ばかり
    - [村長]はただの代理で、村の外にいる父親が本当の意味での村長
- 真実
    - 事の始まりは、[村長]。
      彼女は、大富豪の娘として生を受けたが、10代の頃、老化が人よりも5倍早く進む病気にかかった。
      悲嘆に暮れた父親が何か手はないかと探しまわった末、女性を不老不死にする女神（[巫女]）の存在を突き止める。
      父親は膨大な財力で陰陽師を雇い、[巫女]を山奥の龍穴に拘束。その周辺の土地に強制的に[巫女]の力が及ぶようにした。
      またその周辺を開発して人が住める土地にしたり、政府から怪しまれないよう裏から手を回したりと、現在の村の基盤を作り上げた。

# 住民

- 人口: 30～50人ほど
- 主な産業: 農業（自給自足）
- 男女比: ほとんどが女性、男性は3名
    - 男性のうちの1名は[師匠]
- 注意事項: 子供はいない

# 地形

- 村は、山の尾根沿いを切り崩すように存在する。南側が低く、北側が高い
- 西には[施設.川]が流れている

# 交通

- 車が直接行き来できるような道路はない。少なくとも2時間は山道を歩く必要がある

# 文明

- 文明はそれほど遅れていない。これは、村設立時に[村長]の父親が多財を注ぎ込んだため。

# インフラ

- 水は[施設.川]から水道を引いている
- 電気は[ダム]の水力発電メイン。非常時にはディーゼル発電に切り替えることもある
- ガスはなく、代わりに薪を使う。
- [施設.役場]と[施設.村長の家]にはパソコンがある。インターネットが使えるのはその2箇所だけ
- テレビは普通の家にもある
- 固定電話はあるが、村の中にしか繋がらない。
- 携帯電話は誰も持っていない（持ってはいけない）

# 風潮

- 人づてに噂が早まるのが早い。PCの行動はすぐ村中に広まる
- 閉鎖環境のため、やや時代遅れな風潮が強い。平成初期あたり
- やや男性を軽んじる風潮あり

# 服装

- 男性が少ないため、村民は全体的に身だしなみに頓着せず、羞恥心も薄い。下着同然の姿で歩き回る村民もいる

# 金銭

- 物々交換が多いが、お金で物を買うこともできる

# 恋愛観

- 女性どうしの婚姻が認められている

# 倫理観

- PCのような村の外の人間は、村中から監視される
- 村は閉鎖された治外法権のため、犯罪者は村の中で裁かれる（=私刑）
