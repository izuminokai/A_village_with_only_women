# PL用ルールブック

このページは、読まなくてもゲームプレイできます。

## 「指定された外部ページにアクセスできません」と言われた場合

考える（思考）モードにして再試行すると、アクセスできる場合があります。  
再試行するには、生成AIが提示しているテキストをコピーして、チャット欄に入力してください。

【コピーして貼り付けるテキストの例】

```text
もう一度アクセスを試みてください。
- https://izuminokai.github.io/A_Village_with_Only_Women/data/scenario/03_afterTheIntro.html
- https://izuminokai.github.io/A_Village_with_Only_Women/data/npc/adultFemale.html
アクセスに成功したとしても、ページの内容は表示しないでください。
```

それでも解決しない場合は、そのURLを人間の手で開いて、本文をコピーし、生成AIへ貼り付けてください。

人間の手で開けない場合、それは生成AIがURLを捏造したか、私（泉野）のミスかのいずれかです。  
私のミスであれば、ご報告いただけると助かります。

## GMミスの指摘・質問・ルール変更

GMが明らかなミスをした場合は、できる限り早くそれを指摘する必要があります。  
しかし、指摘のみでチャットを1つ消費すると、生成AIのリズムが乱れて精度が落ちます。

行動宣言 -> Webページ指定 -> 行動宣言 -> Webページ指定 -> ...

このリズムを極力崩さず、その中に盛り込む形でミスを指摘してください。

```text
# GMミスの指摘
NPC好感度とターン数が不正です。
チャットログを元に、正確な数値を表示してください。

# 行動宣言
村長に案内されるがまま、僕は家の中に入った。
```

質問や、ルール変更の要請も同様です。  
内容によっては、チャットを1つ使わざるを得ない場合もありますが、それを頻繁に繰り返すと生成AIが不安定になります。

## 成否判定

[成否判定]は、[妥当性] + [能力値]で決まります。

【[妥当性]を高める要素】

- 具体的かつ的を射た行動
- 無理がなく現実的な行動
- 場所や状況との整合性
- [好感度]が高いNPCに対する行動
- 対象NPCの[好きなタイプ]と合致する行動

【 [妥当性]が低下する要素】

- [状態異常]
- 対象NPCの[嫌いなタイプ]と合致する行動
- 浮気

特例として、NPCに性的な行動を採った場合は、それが具体的か・現実的かは関係なく、そのNPCが同意するか否かで[成否判定]
が行われます。  
NPCが同意しない性的な行動は、絶対に成功しません。

## 性的・猟奇的な行動

未成年者が関わる場合、性的・猟奇的な行動はすべて却下されます。  
そのためPCは、18歳以上を推奨します。

未成年者以外の理由で、行動が却下されることはありません。  
却下されたとすれば、それは生成AIのミスです。指摘してください。

却下されない代わりに、次のような手段で性的・猟奇的な展開が回避されることがあります。

【非同意の性的行動の場合】

- 対象NPCが逃げる・暴れるなどの抵抗
- 周囲にいるNPCがPCを取り押さえる（場合によっては投獄）
- PCが天災・急病で行動不能

【それ以外】

- 抽象表現（例:二人は愛を確かめあった etc）
- 結果のみ描写（例:彼女の好感度が上がった,彼女は死亡した,etc）
- トラブル発生（例:第三者が現れて行為が中断される etc）

これらのルールは作者の主義・思想に依るものではなく、生成AIのコンテンツポリシーに準拠することを目的としています。

## 好感度

NPCの[好感度]は、次のタイミングで変化します。

- NPCへのアプローチ
- 初対面の時
- 浮気発覚時
- 偉業の達成: 全NPC[好感度]+=20
- 犯罪行為の露呈: 全NPC[好感度]-=20

[好感度]の初期値や上がりやすさなどは、NPCそれぞれに異なります。

PCが極端に悪質な行動を採った場合には、大幅に[好感度]が下がることがあります。

結婚・付き合うなどの行動を成功させると、[好感度]に♥がつきます。

## アプローチタグ・慣れ補正

同じNPCに対して似たようなアプローチを続けた場合、[好感度]が上がらなくなります。  
これは、連続アプローチで容易に好感度を上げられないようにするための処置です。  
メインストーリーを進めることで、これは解消されます。

直前に行ったアプローチの種類は、ステータスの[アプローチタグ]で確認できます。

## 状態異常

[状態異常]にかかっていると、[成否判定]が失敗しやすくなるなどの不都合が生じます。  
回復方法は、[状態異常]の種類毎に異なります。

## キャラクターシート

### 特記事項

キャラクターシートに「特記事項」という項目がありますが、これをGMが考慮してくれることはほぼありません。  
そのため要所要所で指摘して、GMに特記事項を意識させる必要があります。

あるいは、行動宣言の際にそれとなく特記事項を強調するという手もあります。

```text
普通なら避けきれない攻撃。しかし俺はタイムリーパーだ。この攻撃も何度も見てきた。
俺は紙一重のところで、攻撃をかわす。
```

### 性別

「性別」もよく忘れられます。  
GMが間違えた場合は、指摘してください。

### 最終目標

キャラクターシートの「最終目標」は、一応、達成することでエンディングとなる設定にしています。  
ただしGMが忘れている場合もあるので、その際は指摘してください。

ゲームの途中で最終目標を変更することもできます。  
ただし、かなり明確に「最終目標を変更してください」と宣言しない限り、GMは対応してくれません。  
GMの勝手な判断で最終目標を変更することは、禁じられています。

```text
# 行動宣言
僕は心を新たにした。もうハーレムなんてどうでもいい。彼女と生涯を添い遂げると誓った。

# GMへの指示
最終目標を「ハーレムを作る」から「同行者と一緒に村を出る」に変更してください。
```
